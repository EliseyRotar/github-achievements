# My GitHub Achievements Checklist

Track your progress here. Check off each item as you earn it.

---

## One-Time Badges

- [ ] **Quickdraw** — Open an issue and close it within 5 minutes
  - Method: Use the Issue Template "Quickdraw Test" or run `./scripts/get-quickdraw.sh`

- [ ] **YOLO** — Merge a PR without a code review
  - Method: Open any PR → add the `yolo` label → auto-merged by workflow

- [ ] **Public Sponsor** — Sponsor any GitHub user/org
  - Method: Go to any GitHub Sponsors page and sponsor for $1/month

---

## Tiered Badges — Pull Shark 🦈

- [ ] Base — 2 merged PRs
- [ ] Bronze — 16 merged PRs
- [ ] Silver — 128 merged PRs
- [ ] Gold — 1024 merged PRs

Method: Run `./scripts/get-pull-shark.sh` with desired count, or manually create+merge branches.

---

## Tiered Badges — Pair Extraordinaire 👥

- [ ] Base — 1 co-authored merged PR
- [ ] Bronze — 10 co-authored merged PRs
- [ ] Silver — 24 co-authored merged PRs
- [ ] Gold — 48 co-authored merged PRs

Method: Run `./scripts/get-pair-extraordinaire.sh` with your co-author's name and email.

---

## Tiered Badges — Galaxy Brain 🧠

- [ ] Base — 2 answers marked as "Answered" in Discussions
- [ ] Bronze — 8
- [ ] Silver — 16
- [ ] Gold — 32

Method:
1. Enable Discussions in this repo's Settings
2. Create Discussion questions
3. Answer them (with another account or a friend) and mark as "Answered"

---

## Tiered Badges — Starstruck ⭐

- [ ] Base — 16 stars on a repo
- [ ] Bronze — 128 stars
- [ ] Silver — 512 stars
- [ ] Gold — 4096 stars

Method: Share this repo! Post it on social media, Reddit, or ask friends to star it.
